# AI editing rules

This is an Overleaf LaTeX project.

## Allowed edits

AI may edit:
- .tex files
- .bib files
- .sty and .cls files
- markdown documentation files

## Rules

- Preserve citation keys unless explicitly asked otherwise.
- Preserve labels, refs, equations, and figure references.
- Do not rewrite large sections without explicit instruction.
- Do not delete comments unless they are clearly obsolete.
- Keep terminology consistent across the document.
- Explain every nontrivial change in the pull request description.

## Forbidden edits

Do not edit generated files:
- .aux
- .log
- .out
- .toc
- .bbl
- .blg
- .synctex.gz
- generated PDFs unless explicitly requested

## Preferred workflow

Create a new branch for each task.
Open a pull request.
Do not push directly to main.****
